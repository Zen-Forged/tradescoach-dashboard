# TradesCoach.ai — HVAC Intelligence Dashboard

A weekly operational intelligence dashboard for HVAC business owners. Surfaces profit leaks, technician utilization, and prioritized actions — built to feel like a coach, not a spreadsheet.

## Tech Stack

- **React 18** + **Vite**
- **Tailwind CSS**
- **Recharts** for data visualization
- Deployed via **GitHub Pages**

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build & Preview

```bash
npm run build
npm run preview
```

## Deploy

Pushes to `main` automatically deploy to GitHub Pages via the included GitHub Actions workflow.

> **V1 Prototype** — static mock data. Backend integration with ServiceTitan, Jobber, and HouseCall Pro planned for V2.
